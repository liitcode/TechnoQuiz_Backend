# TechnoQuiz Backend

[TechnoQuiz Backend](https://technoquiz-env.eba-33dpsiuk.ap-south-1.elasticbeanstalk.com) 


## How to run this project

- Clone Repo 
```git clone https://github.com/liitcode/TechnoQuiz_Backend```

- Install NPM packages
```npm install```

- Add .env file 
```Env
USERNAME_MONGODB = 
PASSWORD_MONGODB = 
CLUSTER_MONGODB = 
DATABASE_MONGODB = 
AUTH_TOKEN_SECRET = 
QAPI_TOKEN = 
QAPI_BASE = 
RAZORPAY_KEY_ID = 
RAZORPAY_KEY_SECRET = 
ENCRYPTION_SALT = 
```

- Run the Project
```npm start```

- You can set the port in index.js file or default will be http://localhost:8000
